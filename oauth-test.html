OAuth Redirect URI Test

Expected URI: http://localhost:3000/auth/callback.html

Test this URI: http://localhost:3000/auth/callback.html

Add EXACTLY this URI to Google Cloud Console:

http://localhost:3000/auth/callback.html

Steps:

  1. Go to Google Cloud Console Credentials
  2. Find your OAuth 2.0 Client ID
  3. Click Edit (pencil icon)
  4. In "Authorized redirect URIs", add the URI above
  5. Remove any old URIs like http://localhost:3000/auth/callback
  6. Click Save
  7. Wait 2 minutes for changes to propagate
